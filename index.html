Discussion Board

This application displays all Topics posted by the users and the real-time count of posts/answers for that specific topic.

  • SignUp/Login account
  • JWT token based authentication
  • New topic creation with name and description and a image.
  • Add picture from various source including Google Image, Flikr, Facebook... (used 'Filepicker' API)
  • Adding a new post in the topic and a image
  • Add a comment for each post in the topic
  • Up/down vote for each post
  • Data is stored in MongoDB
  • Karma/Jasmine Testing has been performed partially
